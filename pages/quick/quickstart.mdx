# Quick Start

Quick start APIs server with all predefined models and data.

## Get files

Get quick start files in the project directory

In project root directory, run following command:

```bash
# For windows
node_modules/.bin/crudsify-cli.cmd copy

# For others
node_modules/.bin/crudsify-cli copy
```

## Migrate tables

Run following command to create/migrate tables in the database.

```bash
npx sequelize-cli db:migrate
```

> **NOTE**: Make sure database configuration should be correctly defined in the **environment** file and exists.

## Seed data

Run following command to seed data in tables.

```bash
npm install @faker-js/faker --save-dev
npx sequelize-cli db:seed:all
```

## Authentication

To enable authentication, set authentication **true** in configuration file, by default authentication is false.

```js filename="config/index.js" showLineNumbers
module.exports = {
  authentication: true,
};
```

and update index.js file as below.

```js filename="index.js" showLineNumbers
const crudsifyServer = require("crudsify");
const { refreshStrategy } = require("./strategies/refresh");
crudsifyServer(refreshStrategy);
```

## Run API server

Run following command to start API server.

```bash
node index.js
```
